# 😄 Dstresshub

A faithful replication  redesigned for your **Laughter Therapy & Stress Relief** business. Pure HTML/CSS/JS inside a Vite project.

---

## 🔑 Design DNA (matching Wix template)

| Element | Detail |
|---|---|
| **Font** | Cormorant Garamond (serif display) + Jost (body) |
| **Palette** | Deep olive-black `#1a1c16` · Warm gold `#c9a84c` · Cream `#f5f0e8` |
| **Layout** | Full-width sections · 2-col grids · Staggered image collages |
| **Scrolling** | Parallax hero + 2 fixed-bg quote strips |
| **Animations** | Scroll reveal · Custom cursor · Hero parallax · Gallery drag |
| **Ticker** | Gold marquee strip between hero and content |
| **Gallery** | Staggered-height draggable horizontal scroll |
